Instruções para execução da aplicação

1) Instalar o node.js.

	-Windows: https://nodejs.org/en/download/
	
	-Linux:  sudo apt-get install node.js
 
2) Extrair os arquivos enviados.

3) Abrir o terminal dentro do diretório onde se encontram os arquivos
   e executar o seguinte comando:
	
	node server.js

4) Abrir duas janelas diferentes no navagador e digitar:

	localhost:3000

   ou, abrir o diretorio 'public' e abrir 'index.html' em duas janelas diferentes.

5) Efetuar o teste.
